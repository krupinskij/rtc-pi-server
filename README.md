# RTC PI

Streaming raspberry pi camera to web browser - server part (NOT FINISHED YET)

## Installation

Download or clone repository

Add .env file

```
CONNECTION_STRING=...
ACCESS_TOKEN_SECRET=...
REFRESH_TOKEN_SECRET=...
PORT=...
ORIGIN=...
```

Run

```bash
# installing dependencies
$ yarn install
```

## Running the app

```bash
# development
$ yarn start
$ yarn dev
```
